# 服务申请

划词翻译支持的部分第三方服务需要在它们各自的官方平台申请之后才能使用。这些服务都提供了足够个人使用的免费额度，且申请过程不会超过 5 分钟。**这些服务的免费额度是由它们各自的官方平台提供的，超出免费额度后产生的费用也是由它们各自的官方平台收取，与划词翻译无关**。请点击左侧菜单栏中感兴趣的服务查看申请步骤。
